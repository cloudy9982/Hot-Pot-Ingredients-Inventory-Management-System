# 關東煮存貨管系統 backend

# 安裝方法 
``` pip install -r requirements.txt ```

# 使用操作
> 說明：`python app.py `後，開另一個terminal，寫`python db.py `，fake deta才能放進去

> frontend connected backend
clone front 後，下載完能正常開啟，操作以下指令

>> backend
` python db.py`
` python app.py`

>> frontend
依照順序操作
`npm install`
`npm run build`
` npm start `

# 檢查語法
`pylint <檔案名稱.py>`

# 檢查unit test
`python -m unittest discover`
