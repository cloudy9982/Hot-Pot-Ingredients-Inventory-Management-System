# 關東煮存貨管系統 backend

# 安裝方法 
``` pip install -r requirements.txt ```

# 使用操作
`python db.py`，fake deta出現在instance資料夾內
`python app.py`，開啟主程式


frontend connected backend
clone front 後，下載完能正常開啟，操作以下指令

> backend
`python db.py`
`python app.py`

> frontend
`npm install`
`npm run build`
`npm start `

# 檢查語法
`pylint <檔案名稱.py>`

# 檢查unit test
`python -m unittest discover`
